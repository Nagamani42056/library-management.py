# Library Management System

## Overview
A console-based Library Management System developed using Python and Object-Oriented Programming (OOP) principles.

## Features
- Add Books
- View Books
- Issue Books
- Return Books
- Manage Library Records

## Technologies Used
- Python
- Object-Oriented Programming (OOP)

## OOP Concepts Applied
- Classes and Objects
- Constructors
- Encapsulation
- Association

## How to Run
1. Clone the repository
2. Open terminal
3. Run:

```bash
python library_management.py
